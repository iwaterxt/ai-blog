3:Tca3,<h1>Attention Is All You Need 解读</h1>
<h2>背景</h2>
<p>2017年，Google 在论文《Attention Is All You Need》中提出了 Transformer 架构，彻底摒弃了 RNN/LSTM 的序列建模方式，仅用 Attention 机制就实现了 SOTA 效果。</p>
<h2>点积注意力</h2>
<p>Attention 的核心是 Scaled Dot-Product Attention：</p>
<p><span class="katex-display" data-math="%5Ctext%7BAttention%7D(Q%2C%20K%2C%20V)%20%3D%20%5Ctext%7Bsoftmax%7D%5Cleft(%5Cfrac%7BQK%5ET%7D%7B%5Csqrt%7Bd_k%7D%7D%5Cright)V"><br/>\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V<br/></span></p>
<p>其中 <span class="katex-inline" data-math="Q%2C%20K%2C%20V">Q, K, V</span> 分别代表 Query、Key、Value 矩阵。</p>
<h3>Q、K、V 的直观理解</h3>
<p>可以这样形象地理解：</p>
<ul><li><strong>Query（查询）</strong>：当前位置想要查询的内容</li>
<li><strong>Key（键）</strong>：每个位置的"索引标签"</li>
<li><strong>Value（值）</strong>：每个位置的原始信息</li>
</ul>
attention 分数就是 Query 和 Key 的匹配程度，然后根据这个分数对 Value 加权求和。
<h2>Multi-Head Attention</h2>
<p>单头注意力只能捕捉一种关联模式。多头注意力将其扩展：</p>
<p><span class="katex-display" data-math="%5Ctext%7BMultiHead%7D(Q%2C%20K%2C%20V)%20%3D%20%5Ctext%7BConcat%7D(%5Ctext%7Bhead%7D_1%2C%20%5Cldots%2C%20%5Ctext%7Bhead%7D_h)W%5EO"><br/>\text{MultiHead}(Q, K, V) = \text{Concat}(\text{head}_1, \ldots, \text{head}_h)W^O<br/></span></p>
<p>其中 <span class="katex-inline" data-math="%5Ctext%7Bhead%7D_i%20%3D%20%5Ctext%7BAttention%7D(QW_i%5EQ%2C%20KW_i%5EK%2C%20VW_i%5EV)">\text{head}_i = \text{Attention}(QW_i^Q, KW_i^K, VW_i^V)</span></p>
<h2>代码实现</h2>
<pre><code class="hljs language-python">import torch
import torch.nn as nn
import torch.nn.functional as F
import math
<p>class MultiHeadAttention(nn.Module):<br/>    def __init__(self, d_model, num_heads):<br/>        super().__init__()<br/>        self.num_heads = num_heads<br/>        self.d_k = d_model // num_heads<br/>        self.W_q = nn.Linear(d_model, d_model)<br/>        self.W_k = nn.Linear(d_model, d_model)<br/>        self.W_v = nn.Linear(d_model, d_model)<br/>        self.W_o = nn.Linear(d_model, d_model)</p>
<p>def forward(self, x):<br/>        B, T, C = x.size()<br/>        q = self.W_q(x).view(B, T, self.num_heads, self.d_k).transpose(1, 2)<br/>        k = self.W_k(x).view(B, T, self.num_heads, self.d_k).transpose(1, 2)<br/>        v = self.W_v(x).view(B, T, self.num_heads, self.d_k).transpose(1, 2)</p>
<p>scores = torch.matmul(q, k.transpose(-2, -1)) / math.sqrt(self.d_k)<br/>        attn = F.softmax(scores, dim=-1)<br/>        out = torch.matmul(attn, v)</p>
<p>return self.W_o(out.transpose(1, 2).contiguous().view(B, T, C))</code></pre></p>
<h2>总结</h2>
<p>Transformer 的成功证明了：<strong>全局建模 + 可并行 = 强大且高效</strong>。Attention 机制让模型能够自由地建立任意位置之间的依赖关系，这是 RNN 的最大局限。</p>
